Este proyecto implementa una aplicación en Python que permite:
- Crear un árbol con raíz
- Agregar hijos a un nodo
- Mostrar la estructura del árbol
- Realizar recorridos (preorden, inorden, postorden)
- Calcular la altura del árbol
- Verificar si el árbol es binario
